验证码适合用 Redis 的原因：

  • 有天然过期时间
  • 不需要永久保存
  • 读取速度快
  • 登录校验完成后可以删除

Token 登录态适合用 Redis 的原因：

  • 可以统一控制过期时间
  • 退出登录时直接删除 key
  • 后续集群部署时，多台后端都能共享登录状态
