这篇文章记录我把黑马点评 Java 后端改造成 Go 后端的前半段过程。

在真正写登录拦截器、点赞、Feed 流这些功能之前，我先完成了这些基础工作：

  • 从零搭建 Go 后端项目骨架
  • 读取 configs/config.yaml
  • 连接 MySQL 和 Redis
  • 设计 Handler、Service、Repository 分层
  • 统一前端响应结构 Result
  • 实现店铺分类、店铺列表、店铺详情、优惠券列表、博客详情、用户信息等基础接口

这篇文章更像一次阶段复盘：它不是只讲某一个接口，而是把项目从“空目录”推进到“前端首页和登录链路能跑起来”的全过程串起来。
